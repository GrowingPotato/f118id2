Задание:
Даны два CSV‑файла, содержащие данные об успеваемости студентов.

Первый файл — список студентов, содержит столбцы:
id, ФИО.

Второй файл — оценки по предметам, содержит столбцы:
id_студента, предмет, оценка.
Связь между файлами осуществляется по id (номеру студенческого билета).

Требуется:

Считать данные из обоих файлов и организовать их хранение в удобных структурах данных (например, списки словарей или словари списков).

Реализовать функции:

average_by_subject(data) — вычисляет среднюю успеваемость по каждому предмету.

average_by_student(data) — вычисляет среднюю успеваемость для каждого студента.

best_student(data) — определяет студента с наилучшей общей успеваемостью.
